# Portfolio Structure

## Folder Organization

```
portafolio/
├── index.html              # Profile/About page
├── work.html               # Work Experience page
├── projects.html           # Applied/Industrial projects
├── research.html           # Research/Scientific projects
├── css/
│   └── style.css           # Main stylesheet
├── js/
│   └── animations.js       # Scroll animations
├── assets/
│   └── images/
│       └── cover_2.png     # Profile image
├── Projects/               # Content files for projects & research
└── Work/                   # Content files for work experience
```

## Notes

- All `explanation_*.pdf` files are **NOT** included in the portfolio
- All `.exe` files are hosted externally on Google Drive (not in repository)
- File names use snake_case with descriptive prefixes:
  - `animation_` for animated content (GIF/MP4)
  - `photo_` for static images (PNG)
